# Flight-Information-analysis-Dashboard-PowerBI
data analytics with Power BI and built a Flight Information Analysis Dashboard to transform raw aviation data into actionable insights.

This dashboard focuses on understanding airline performance, passenger metrics, and booking behaviors.

Key Insights & Features:

Airline Performance: Analyzed passenger volumes and flight distributions across multiple carriers, identifying key leaders in volume.

Booking Trends: Created a distribution breakdown of ticket booking statuses (Confirmed, Cancelled, and Pending) to visualize operational health.

Advanced Drill-downs: Implemented a decomposition tree to map out the exact distribution of flights based on specific destinations and airlines.

User Interactivity: Added custom airline slicers and interactive filtering to allow stakeholders to drill down into specific carrier details instantly.
